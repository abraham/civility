GMR
===

GMR is the easiest way to manage your Civ5 hotseat games hosted by [http://multiplayerrobot.com/](http://multiplayerrobot.com/).

Install
-------

    $ gem install gmr


Usage
-----

Authenticate yourself:

    $ gmr auth token

_Run `gmr auth` if you don't know where to get your GMR token._

Get a list of your games:

    $ gmr games


Download a save file to play:

    $ gmr play game_name

Open Civ5, play your hotseat turn, and save to the same file.

Upload you completed turn:

    $ gmr complete game_name

OS Support
----------

gmr has only been minimally tested on OS X.
